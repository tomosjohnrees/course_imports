# What This Course Covers

This course takes you from zero to confident with GitHub Actions across 12 topics. Here is the roadmap:

1. **Introduction** (this topic) — What GitHub Actions is, core concepts, and how it fits into CI/CD.
2. **Your First Workflow** — Write, commit, and run a real workflow file from scratch.
3. **Triggers and Events** — Master the `on` key: push, pull_request, schedule, workflow_dispatch, and more.
4. **Jobs and Runners** — Configure parallel and dependent jobs, choose runner environments, and understand hosted vs. self-hosted runners.
5. **Steps and Actions** — Write multi-step jobs, use Marketplace actions, and create your own composite actions.
6. **Variables and Secrets** — Work with environment variables, repository secrets, and configuration contexts.
7. **Expressions and Conditionals** — Use `if` conditions, status functions, and expression syntax to control workflow logic.
8. **Build and Test** — Set up real CI pipelines for common languages and frameworks.
9. **Deployment** — Deploy to cloud platforms, manage environments, and implement approval gates.
10. **Matrix Strategies** — Test across multiple OS/language/version combinations in a single workflow.
11. **Reusable Workflows** — Share workflow logic across repositories with reusable workflows and composite actions.
12. **Security Best Practices** — Pin action versions, manage permissions with `GITHUB_TOKEN`, and protect against common CI/CD attack vectors.

By the end, you will have the skills to design, build, and maintain production-grade CI/CD pipelines with GitHub Actions.
