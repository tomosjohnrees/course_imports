# Lists and Links

## Lists

Unordered lists use `-`, `*`, or `+` as bullet markers. Indent with two spaces to create nested items.

Ordered lists use numbers followed by a period. The actual numbers don't matter — Markdown will renumber them automatically.

## Links

Links are written as `[display text](url)`. For example:

```
[Markdown Guide](https://www.markdownguide.org)
```
