# Welcome to the Course Authoring Guide

Course Imports is a desktop application for importing and viewing educational courses. It takes a structured folder of files — JSON metadata and markdown content — and renders them as an interactive learning experience, complete with syntax-highlighted code examples, callout boxes, and quizzes.

This course will teach you how to author your own courses from scratch. By the end, you'll know how to lay out a course folder, write metadata, create rich content blocks, build quizzes, and validate everything before loading it into the app.
