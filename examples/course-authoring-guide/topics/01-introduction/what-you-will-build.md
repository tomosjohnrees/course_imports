# What You'll Build

Over the next seven topics, you will build a complete course from an empty folder to a fully validated, loadable package. Here is what each topic covers:

1. **Introduction** (this topic) — understand what Course Imports is and what a course looks like at a high level.
2. **Folder Structure** — learn the exact directory layout the app expects and the naming conventions for topic folders.
3. **Course Metadata** — write a `course.json` file with all the required fields.
4. **Content Blocks** — create text, code, image, and callout blocks inside a topic's `content.json`.
5. **Writing Quizzes** — add multiple-choice quizzes to test your learners.
6. **Building a Course** — put everything together into a complete, multi-topic course.
7. **Validation and Loading** — run through the validation checklist and load your course into the app.

By the end, you will have authored a working course that you can import and view in Course Imports.
