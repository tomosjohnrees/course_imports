# Building a Course End to End

You have learned each piece of the puzzle individually: folder structure, course metadata, content blocks, and quizzes. Now it is time to put them all together.

In this topic, you will walk through creating a complete, working course from scratch. The example course is called **Intro to Git** and contains two topics:

1. **What Is Git?** — a brief explanation with a code example and a quiz.
2. **Making Commits** — a walkthrough of staging and committing changes, with a quiz.

By the end, you will have seen every file you need to create, in the exact order you would create them. You can use this as a template whenever you start a new course.
