# Step 3: Create the First Topic's Content

Now comes the interesting part: writing the `content.json` for the first topic, `01-what-is-git`. This file is a JSON array of blocks that the app will render from top to bottom.

For this topic, we will use four block types:
- A **text** block (referencing an external markdown file) to introduce Git.
- A **code** block showing a terminal command.
- A **callout** block with a helpful tip.
- A **multiple-choice quiz** to check understanding.
