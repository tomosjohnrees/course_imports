# What Is Git?

Git is a distributed version control system that tracks changes to files over time. It lets you:

- **Save snapshots** of your project at any point (called "commits").
- **Branch** to work on new features without affecting the main codebase.
- **Collaborate** with others by merging changes together.
- **Revert** to earlier versions if something goes wrong.

Git is the most widely used version control system in the world. Nearly every open-source project and most professional software teams rely on it daily.

In this topic, you will learn how to check whether Git is installed and see its version number.
