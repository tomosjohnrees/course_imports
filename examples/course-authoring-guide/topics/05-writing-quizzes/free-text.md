# Quiz Tips

Here are some tips for writing effective quizzes:

- **Write clear questions.** The learner should understand exactly what is being asked without ambiguity.
- **Use plausible distractors.** Wrong options should sound reasonable so the quiz genuinely tests understanding, not just elimination.
- **Vary the correct answer position.** Don't always put the correct answer first or last — spread it across different indices.
- **Add explanations.** The optional `"explanation"` field is a great way to reinforce learning after the learner answers, whether they got it right or wrong.
