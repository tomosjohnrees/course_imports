# Topic Completion Rules

How quizzes affect topic completion is an important detail for course authors.

- **Topics without any quizzes** are marked complete as soon as the learner opens them. Simply viewing the content is enough.
- **Topics that contain one or more quizzes** are only marked complete when the learner has answered every quiz in that topic.

This means that adding even a single quiz to a topic changes its completion behaviour. Keep this in mind when deciding where to place quizzes — if a topic is purely informational and you want learners to move through it quickly, leave out the quizzes. If you want to ensure engagement, add at least one.
