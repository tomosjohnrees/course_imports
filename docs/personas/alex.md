# Persona: Alex Kim — Power User / Hobbyist Collector

## Demographics

- **Age:** 24
- **Occupation:** Graduate student (computational biology)
- **Location:** Toronto, Canada
- **Technical comfort:** High — comfortable with git, terminal, scripting; uses multiple dev tools daily

## Background

Alex is a voracious learner who has loaded dozens of courses into Course Imports on topics ranging from Rust programming to music theory to philosophy of science. They treat the app like a personal library of structured knowledge. They care about efficiency — keyboard navigation, fast loading, dark mode — and push the app to its limits with large courses and rapid switching between them.

## Goals

- Maintain a large personal library of courses across many subjects
- Switch between courses quickly via Recent Courses
- Navigate entirely by keyboard when possible
- Use dark mode exclusively (late-night study sessions)
- Load courses from both GitHub and local folders interchangeably

## Typical workflows

1. Discovers new courses on GitHub, Reddit, or Discord communities
2. Loads courses from GitHub URLs; also keeps a local folder of courses they've cloned
3. Switches between 3–4 active courses in a given week
4. Uses keyboard arrows to navigate between topics quickly
5. Skims text blocks, focuses on code blocks and quizzes
6. Configures a GitHub personal access token immediately to avoid rate limits
7. Uses dark mode and expects it to look polished everywhere

## Pain points and frustrations

- Recent Courses list getting cluttered — wants to manage or remove entries
- Loading the same GitHub course again should be fast (some caching would help)
- Dark mode inconsistencies are immediately noticeable and annoying
- Large courses (50+ topics) should still feel snappy in the sidebar
- Expects keyboard shortcuts to work consistently — gets frustrated if focus gets trapped

## What success looks like

Alex has 20+ courses in their Recent Courses list, switches between them in seconds, navigates topics entirely by keyboard in dark mode, and the app stays responsive and visually consistent throughout.
