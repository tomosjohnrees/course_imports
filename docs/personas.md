# Personas

User personas for testing and evaluating Course Imports from different perspectives.

| Persona | Description |
|---|---|
| [Sarah](personas/sarah.md) | Casual self-directed learner. Moderate technical comfort. Loads courses from shared links, works through topics at her own pace, relies on progress tracking. |
| [Marcus](personas/marcus.md) | Course author and senior developer. High technical comfort. Creates courses in VS Code, previews locally, publishes via GitHub. |
| [Priya](personas/priya.md) | AI-assisted content creator. Moderate-to-high technical comfort. Uses LLMs to generate courses, loads them locally to review, iterates quickly on fixes. |
| [James](personas/james.md) | Community college instructor. Moderate technical comfort. Distributes course material to 30+ students via GitHub URLs, needs reliability and zero tech support. |
| [Elena](personas/elena.md) | Non-technical learner. Low technical comfort. Received a course link from a relative, no GitHub knowledge, needs everything to just work. |
| [Alex](personas/alex.md) | Power user and hobbyist collector. High technical comfort. Dozens of courses loaded, keyboard navigation, dark mode, pushes the app to its limits. |
